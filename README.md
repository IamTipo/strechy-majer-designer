# STŘECHY MAJER Designer v2.5.0

Kladečský plán pro střešní krytinu PREFA R.16.

## Struktura

- `index.html` – HTML kostra aplikace
- `css/style.css` – vzhled aplikace
- `js/app.js` – aplikační logika
- `images/` – logo, QR, fotografie PREFA R.16 a favicon
- `data/version.json` – informace o verzi

## Nasazení na GitHub Pages

Nahraj celý obsah této složky do kořene repozitáře `strechy-majer-designer`.
Soubor `index.html` musí zůstat v hlavní složce repozitáře.
